# RF Electrical Power System

An RF-based electrical power transmission and control project designed for experimental and educational purposes.

This project demonstrates wireless power concepts combined with electrical circuit design and simulation.

---

## Overview

The RF Electrical Power System explores the fundamentals of radio-frequency (RF) energy transmission and power control.

The project includes:

- RF signal generation
- Power transmission concept
- Receiver-side electrical conversion
- Circuit simulation using Proteus

This project is intended for learning RF principles and electrical power transfer behavior.

## Working Principle

1. RF transmitter generates high-frequency signal.
2. Signal propagates wirelessly.
3. Receiver circuit captures RF energy.
4. Rectification and conditioning stage converts RF to usable DC.
5. Output stage delivers power to load.
